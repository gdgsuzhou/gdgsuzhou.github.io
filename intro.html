我们是来自位于中国长江三角洲、具有两千五百多年历史、经常被称为东方威尼斯的美丽的花园城市苏州市的开发者们、企业经理们、高校教师们和大学生们。我们爱好学习和利用各种谷歌开发技术，去开发基于互联网和移动应用平台上的各种新产品和新型商业服务和创新。 我们是一个开放和自由的社区，任何人都可以来加入。我们的目标是通过举办各种互相帮助和分享的活动，交流开发心得和经验，学习各种谷歌开发技术、开源代码技术、公开网络标准技术等等，来推动互联网应用和移动应用的创新和创业。 欢迎你来加入我们这个开放的社区！

GDG是什么

Google Developer Groups (GDG) 面向对 Google 开发者技术感兴趣的开发者；涵盖的内容包括 Android、Chrome、Drive和 Google Cloud 平台，以及各种产品 API (如Cast API、地图 API 和 YouTube API) 等等。

GDG的特征

  • 由开发者社群中的个人爱好者运作
  • 供开发者了解 Google 的相关技术和工具。
  • 可了解当地公司和开发者正在利用这些技术做什么
  • 重点关注开发者和培训技术内容
  • 对公众开放，成员关系公开
  • 能够认识许多机智又风趣的技术人员 :)

GDG没有以下特征

  • 由公司运作
  • 总是充斥着各种推销信息
  • 重点关注最终用户或消费者内容
  • 封闭式网上论坛
