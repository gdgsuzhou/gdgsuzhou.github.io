Google Developers Study Jams

image
| 打架报名中 | Android 基础课学习班 | 活动时间：2015年2月1日开始

活动: Android 基础课学习班

目标: 提高开发者社区的整体技术水平

时间: 2015年2月1日起，错开春节，妇女节，清明节，8个周日, 13:30

地点: 线下，高博国际学院A104室，仁爱路1号; 线上, Udacity, Google+ Hangouts (On Air)

Google Developers Study Jams, 代号：果酱学习班. jams 在这里的本意是大家聚在一起，同时也可以用来指果酱——正巧, Android 操作系统的每一个版本都是用一种小甜点来命名的……

果酱学习班由一系列的、和国际接轨的、社区自发组织的、需要你亲身参与的课程组成。果酱第一课，是 Google 和 Udacity 合作制作的, Android 基础课。这一系列课程将由有资质的 Google Developer Group (GDG) 讲师带领，历时7个星期，每星期一次面对面、肩并肩的学习活动。

这是一个高级课程，需要学员具有3年 Java 编程经验，或其他面向对象编程语言经验，例如 C++, Objective C, 或者 Python. 如果你想温习一下 Java 技能, Udacity 的 Intro to Java 课程正好适合, http://www.udacity.com/course/cs046

为什么一定要报名？Android 是 Google 的开源项目，其应用平台，手机和平板已经司空见惯。在全世界人手一部 Android 设备之前，其普及度会保持飞速增长。现在又多了腕表、眼镜、电视、车载等新的应用平台。课程内容是 Google 制作的。除了能获得权威的教学内容和额外的奖励，你还可以扩展专业领域的人脉，表达你对 Google 技术，对开放技术，对互联网行业，和对苏州谷歌开发者社区的关注！

课程结构:

  • 线上: https://www.udacity.com/course/ud853
  • 线下: 8场活动，每周一场；每节课结束后一场，共7课；第8场，分享大作业(final project)

奖励:

  • 拼图贴纸7张， 每场活动得到不同的一张， 参加全部７ 场活动才能凑齐整幅拼图
  • 学员可以参加 Udacity 课程，并得到讲师和其他学员的指导
  • Udacity 证书，通常参加 $200 的付费课程才有机会获得。Google 将根据参加课程的人数，从 Udacity 购买有限数量的证书，免费发给成绩优秀者。其他成绩合格者可以获得 $160 折扣价。
