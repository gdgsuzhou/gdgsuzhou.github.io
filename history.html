Google Developers Study Jams

image
| 打架报名中 | Android 基础课学习班 | 活动时间：2015年2月1日开始

活动: Android 基础课学习班

目标: 提高开发者社区的整体技术水平

时间: 2015年2月1日起，3月31日止，期间的7个周日上午，可以通过报名表提出你对具体时间的期望

地点: 线下， 独墅湖高教区(高博， 或者西交利物浦大学， 具体地点待定， 可以通过报名表推荐你喜欢的地点); 线上， Google+ Hangouts (On Air, YouTube 直播 + 录像) 报名 (全球统一): http://www.googledevelopersstudyjams.com/

Google Developers Study Jams, 非正式简称： 果酱学习班。 尽管 jams 在这里表示大家聚集在一起， 同时也可以用来指果酱——正巧, Android 操作系统的每一个版本都是用一种小甜点来命名的……

好吧， 果酱学习班是一成串的、 免费的、 全球的、 社区自发组织的， 需要你亲自参与的学习班。 果酱第一课， 是 Google 和 Udacity 合作制作的， Android 基础课程。 这一系列课程将由获得认证的 Google Developer Group (GDG) 讲师带领，历时7个星期，每个星期都会有一次见面活动。

参加此课程需要具备的技能： 这是一个高级课程， 需要学员具有3年 Java 编程经验，或其他面向对象编程语言经验， 例如 C++, Objective C, 或者 Python. 如果你想温习 一下 Java 技能, Udacity 的 Intro to Java 课程正好适合, http://www.udacity.com/course/cs046

学习班的目标: 提高开发者社区的整体技术熟练度

课程结构: 7场活动，约每周一场；每节课结束后一场，共6课；第7场，分享大作业(finalproject)。

学员奖励:

  • 拼图贴纸7张， 每场活动得到不同的一张， 参加全部７ 场活动才能凑齐整幅拼图
  • 完成全部学习过程，并达到如下条件，可以得到“Google 课程毕业”证书。
    • 完成了 Udacity 的课程（ 根据 Google 的后台数据） ， 并且参加了 学习 班的活动（ 根据讲师的记录）
    • 证书上面有“Google Developers Study Jam”的品牌标志
    • 学员可以参加 Udacity 课程(https://www.udacity.com/course/ud853)的免费版， 可以得到讲师和其他学员的指导
    • Google 正在和 Udacity 协调， 给完成免费课程的学员一个折扣价， 以获得参加收费课程才能获得的证书。
    • 有限数量的免费证书将颁发给大作业合格者； 最终多少人能够得到免费证书， 取决于合格人数。

报名方法

这次活动由 Google 统一管理世界各地的学员报名 。 开课之前, Google 会为报名的学员就近分配到所在城市的 GDG.

欢迎关注活动通知原始文档, http://goo.gl/7zXwFJ (会员可评论), http://t.cn/RZfCi6I (只读), 和文当末尾所列的其他网络平台， 以获得最新信息。
